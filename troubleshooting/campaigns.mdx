Soluções rápidas para problemas comuns em campanhas e dicas de otimização de desempenho.

## A Campanha Não Inicia

### Número de Telefone Não Configurado
**Erro:** "Your agent does not have a phone number setup"

**Solução:**
1. Acesse as [Configurações do Assistente](/ai-assistants/settings/general#phone-number-configuration)
2. Configure um número de telefone (Número da plataforma, SIP ou Caller ID)
3. Salve as configurações do assistente
4. Tente iniciar a campanha novamente

### Saldo Insuficiente
**Erro:** "Insufficient balance to start the campaign"

**Solução:**
1. Verifique o saldo da conta na área de cobrança
2. Adicione créditos à conta
3. Retorne à campanha e clique em iniciar

### Nenhum Lead para Ligar
**Erro:** "No leads to call"

**Solução:**
1. Adicione leads manualmente ou importe via CSV
2. Verifique se os leads possuem números de telefone válidos
3. Confirme se os leads não estão todos marcados como concluídos
4. Garanta que a opção "Mark complete when no leads" esteja adequada à sua necessidade

## Nenhuma Chamada Sendo Realizada

### Fora do Horário Permitido
**Problema:** A campanha aparece como ativa, mas nenhuma chamada ocorre

**Verifique:**
- O horário atual está dentro do intervalo permitido
- O dia atual está habilitado para chamadas
- O fuso horário do assistente está configurado corretamente

**Solução:** Ajuste os horários permitidos ou aguarde a próxima janela válida

### Todos os Leads Já Foram Chamados
**Problema:** A campanha parece ociosa

**Verifique:**
- Todos os leads primários atingiram o número máximo de tentativas
- Contatos secundários já foram tentados
- Status de conclusão por objetivo, se estiver usando retry baseado em metas

**Solução:** Adicione novos leads ou revise as configurações de retry

## Problemas de Qualidade de Chamada

### Alta Taxa de Falha
**Sintomas:** Muitas chamadas marcadas como "Failed"

**Causas comuns:**
- Números inválidos na lista de leads
- Problemas de conectividade de rede
- Restrições da operadora

**Soluções:**
1. **Higienize os dados:** Remova números inválidos
2. **Teste chamadas individuais:** Ligue manualmente para alguns leads
3. **Verifique o formato:** Use o padrão correto (+551145678900)
4. **Contate o suporte:** Para problemas técnicos persistentes

### Baixa Taxa de Atendimento
**Sintomas:** A maioria das chamadas cai na caixa postal ou não é atendida

**Otimização:**
1. **Ajuste os horários de chamada:** Teste janelas diferentes
2. **Revise o intervalo de retry:** Espaçe melhor as tentativas
3. **Avalie a qualidade dos leads:** Confirme se os números estão atualizados
4. **Caller ID profissional:** Garanta uma identificação adequada

## Problemas de Lógica de Retry

### Muitas Tentativas
**Problema:** Leads reclamam de excesso de chamadas

**Soluções:**
1. **Reduza o máximo de retries:** De 5 para 2–3 tentativas
2. **Aumente o intervalo:** Espaçe chamadas em 2–4 horas
3. **Desative retry em voicemail:** Considere voicemail como contato válido
4. **Revise variáveis de objetivo:** Garanta rastreamento correto

### Poucas Tentativas
**Problema:** Perda de contatos potenciais por poucas tentativas

**Soluções:**
1. **Aumente o máximo de retries:** Até 5 tentativas
2. **Ative retry em voicemail:** Continue até contato humano
3. **Ajuste intervalos:** Teste diferentes horários do dia
4. **Use retry baseado em objetivo:** Para prospects de alto valor

## Problemas de Conclusão por Objetivo

### Variável de Objetivo Não Funciona
**Problema:** Retry até concluir objetivo não está funcionando

**Requisitos:**
1. **Variáveis pós-chamada** devem estar configuradas no assistente
2. **Variável booleana** deve existir no schema pós-chamada
3. **Nome da variável** deve corresponder exatamente ao da campanha
4. **Extração da IA** deve estar funcionando corretamente

**Solução:** Teste a extração de variáveis pós-chamada com ligações manuais

### Conclusão Falsa de Objetivo
**Problema:** Leads marcados como concluídos incorretamente

**Depuração:**
1. **Revise transcrições** para erros de extração
2. **Teste definições de variáveis** com chamadas de exemplo
3. **Ajuste o prompt da IA** para critérios mais claros
4. **Refine descrições das variáveis** nas configurações do assistente

## Otimização de Desempenho

### Progresso Lento da Campanha
**Causas:**
- Intervalos de retry conservadores
- Horários de chamada limitados
- Alta taxa de falhas reduzindo leads disponíveis

**Soluções:**
1. **Processamento paralelo:** Crie várias campanhas menores
2. **Otimize horários:** Utilize horários de maior taxa de atendimento
3. **Limpe dados:** Remova números problemáticos
4. **Amplie a janela:** Estenda os horários permitidos

### Problemas de Recursos do Sistema
**Sintomas:** Atrasos nas chamadas, gargalos de processamento

**Soluções:**
1. **Reduza campanhas simultâneas:** Limite campanhas ativas
2. **Escalone inícios:** Evite iniciar várias campanhas ao mesmo tempo
3. **Contate o suporte:** Para problemas de performance da plataforma

## Problemas de Gestão de Leads

### Contatos Duplicados
**Problema:** A mesma pessoa recebe várias chamadas de leads diferentes

**Prevenção:**
1. **Limpeza de dados:** Remova duplicados antes da importação
2. **Validação de números:** Verifique variações de formato
3. **Deduplicação:** Utilize ferramentas para identificar duplicatas

### Confusão com Contatos Secundários
**Problema:** Não fica claro qual contato foi atendido

**Solução:**
1. **Nomenclatura clara:** Use nomes descritivos para contatos secundários
2. **Notas de chamada:** Revise transcrições para identificar quem atendeu
3. **Organização dos leads:** Agrupe contatos relacionados corretamente

## Mensagens de Erro Comuns

### "Campaign is already in progress"
**Solução:** A campanha já está em execução. Pause-a para realizar alterações.

### "Campaign is not running"
**Solução:** A campanha está pausada ou encerrada. Inicie para retomar as chamadas.

### "Assistant timezone not set"
**Solução:** Configure o fuso horário do assistente antes de criar a campanha.

## Obtendo Ajuda

### Antes de Contatar o Suporte
1. **Verifique mensagens de erro** para orientações específicas
2. **Revise configurações da campanha** em busca de problemas óbvios
3. **Teste com um único lead** para isolar a falha
4. **Confira status da conta** e cobrança

### Informações a Fornecer
- ID e nome da campanha
- Mensagens de erro recebidas
- Passos realizados antes do problema
- Comportamento esperado vs. atual
- Configuração do assistente e do número

<Warning>
**Atrasos de campanha:** Aguarde de 10 a 15 minutos após iniciar antes de esperar chamadas. O sistema precisa de tempo para inicializar e enfileirar os leads.
</Warning>
