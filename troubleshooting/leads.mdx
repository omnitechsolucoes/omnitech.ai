Soluções rápidas para problemas comuns de gerenciamento de leads e importação.

## Problemas na Importação de Leads

### Falha Total na Importação
**Erro:** "Import failed" ou "File format not supported"

**Soluções:**
1. **Verifique o formato do arquivo:** Utilize apenas CSV ou Excel
2. **Confirme a campanha selecionada:** Garanta que uma campanha esteja escolhida para importação
3. **Limite de tamanho:** Mantenha arquivos abaixo de 10MB para melhor desempenho
4. **Problemas de codificação:** Salve o CSV com codificação UTF-8

### Nenhum Lead Importado
**Problema:** A importação conclui, mas nenhum lead aparece

**Soluções:**
1. **Cabeçalhos das colunas:** Verifique se os títulos correspondem ao formato exigido (phone, name, etc.)
2. **Validação de dados:** Confirme se os números estão no formato correto
3. **Linhas vazias:** Remova linhas em branco do arquivo
4. **Campos obrigatórios:** Certifique-se de que a coluna de telefone contém dados válidos

### Importação Parcial
**Problema:** Alguns leads foram importados, outros falharam

**Soluções:**
1. **Baixe o relatório de erro:** Analise as linhas com falha
2. **Formato internacional:** Use padrão +1234567890
3. **Remova duplicados:** Elimine números repetidos
4. **Dados inválidos:** Corrija números mal formatados ou campos obrigatórios ausentes

### Atraso no Processamento
**Problema:** A importação fica em processamento por muito tempo

**Explicação:**
- Arquivos grandes são processados em segundo plano
- O tempo aumenta conforme o tamanho do arquivo
- O sistema pode enfileirar múltiplas importações

**Soluções:**
1. **Aguarde a conclusão:** Verifique notificações
2. **Divida arquivos grandes:** Separe em lotes de 500–1000 leads
3. **Contate o suporte:** Caso ultrapasse 30 minutos

## Problemas no Gerenciamento de Leads

### Lead Não Aparece na Campanha
**Verifique:**
- Status do lead ("Created" ou "Scheduled")
- Filtros aplicados na campanha
- Se foi marcado como concluído acidentalmente

**Soluções:**
1. **Redefinir status:** Alterar para "Created"
2. **Verificar campanha atribuída**
3. **Atualizar a página**

### Números Inválidos
**Formato correto:** +1234567890  
**Formato incorreto:** (123) 456-7890, 123-456-7890, 1234567890

**Soluções:**
1. **Padronize formato internacional**
2. **Remova espaços e caracteres especiais**
3. **Adicione código do país**
4. **Use ferramentas de validação**

### Leads Duplicados
**Prevenção:**
1. Limpeza de dados antes da importação
2. Verificar variações de formato
3. Identificar duplicados por nome + telefone
4. Usar ferramentas de deduplicação

### Contatos Secundários Não Funcionam
**Requisitos:**
- Número válido no contato secundário
- Contato primário deve atingir limite de tentativas
- Campanha configurada para usar contatos secundários

**Soluções:**
1. Verificar configuração
2. Confirmar status do primário
3. Conferir ajustes da campanha

## Problemas com Variáveis e Dados

### Erro no Mapeamento de Variáveis
**Soluções:**
1. Cabeçalhos devem corresponder exatamente às [configurações do assistente](/ai-assistants/settings/prompt-and-tools#call-variables)
2. Verificar tipos de dados
3. Incluir todas as variáveis obrigatórias
4. Conferir valores padrão

### Informações Ausentes
**Soluções:**
1. Verificar mapeamento de colunas
2. Confirmar inclusão de campos opcionais
3. Validar dados de origem
4. Reimportar se necessário

## Problemas de Integração com Campanha

### Status Incorreto
**Significados:**
- **Created:** Pronto para chamada
- **Scheduled:** Agendado para próxima tentativa
- **Processing:** Em chamada
- **Completed:** Finalizado
- **Max Retries:** Limite atingido

**Soluções:**
1. Alterar status manualmente
2. Reiniciar campanha
3. Ajustar configurações de retry

### Variáveis Não Funcionam na Chamada
**Soluções:**
1. Verificar variáveis nas [configurações do assistente](/ai-assistants/settings/prompt-and-tools#call-variables)
2. Conferir correspondência exata de nomes
3. Validar formato dos dados
4. Testar chamadas manuais

## Problemas com Formato de Arquivo

### CSV
**Erros comuns:**
- Delimitador incorreto (usar vírgula)
- Falta de cabeçalhos
- Aspas não escapadas
- Codificação incorreta

**Soluções:**
1. Usar vírgula como separador
2. Incluir cabeçalhos na primeira linha
3. Usar aspas duplas
4. Salvar como UTF-8

### Excel
**Erros comuns:**
- Múltiplas abas (apenas a primeira é importada)
- Células mescladas
- Fórmulas em vez de valores
- Formatação oculta

**Soluções:**
1. Utilizar apenas a primeira aba
2. Desmesclar células
3. Converter fórmulas em valores
4. Limpar formatação

## Obtendo Ajuda

### Antes de Contatar o Suporte
1. Baixe o relatório de erro
2. Verifique formato do arquivo
3. Teste com lote pequeno (5–10 leads)
4. Consulte a documentação

### Informações a Fornecer
- Mensagens de erro
- Amostra do arquivo (sem dados sensíveis)
- Configuração da campanha
- Comportamento esperado vs. real
- Configuração de variáveis do assistente
