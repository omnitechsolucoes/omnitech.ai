A plataforma permite conectar **três tipos diferentes de números telefônicos** aos seus assistentes de IA:

1. **Números Normais (Dedicados)**

   - Alugados diretamente pela plataforma.
   - Podem ser usados para chamadas **inbound** e **outbound**.
   - Normalmente possuem cobrança mensal e permitem múltiplas chamadas simultâneas utilizando um único número.

2. **Integração SIP (VOIP / PBX)**

   - Conecte um número existente de VOIP ou PBX via SIP.
   - Permite manter seus números atuais e utilizá-los com os recursos de IA da plataforma.
   - Funciona para **inbound e outbound** — consulte o [Guia de Integração SIP](/provisioning/sip-trunking/sip-integration) para instruções de configuração.
   - A configuração é feita diretamente no dashboard, na seção de Números Telefônicos.

3. **Caller ID (Somente Outbound)**

   - Ideal quando você já possui um número telefônico (pessoal ou empresarial), mas não deseja portar ou integrar via SIP.
   - É necessário validar a posse do número (via OTP ou chamada de verificação) para que ele apareça como identificador nas chamadas.
   - **Chamadas inbound não serão atendidas pela IA** quando se utiliza apenas Caller ID — ele serve exclusivamente para identificar chamadas outbound.

## Escolhendo a Melhor Opção

- Se você precisa de **inbound** e não possui uma solução VOIP, opte por um **número dedicado**.
- Se já possui um **PBX** ou provedor VOIP, utilize **integração SIP**.
- Se o objetivo é apenas realizar **chamadas outbound** exibindo seu número atual, escolha **Caller ID**.
