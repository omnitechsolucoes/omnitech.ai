Ao iniciar uma campanha outbound, você pode definir qual número será exibido no telefone do destinatário.

## 1. Número Dedicado

- Se você alugou um número local ou toll-free, pode selecioná-lo como **Phone Number** nas [configurações do assistente](/ai-assistants/settings/general#phone-number-configuration).
- Ideal para **consistência de marca** ou quando o mesmo número também recebe chamadas inbound.
- **Não é necessária verificação por OTP** para números alugados diretamente pela Autocalls.ai.

## 2. Número Próprio Verificado (Apenas Caller ID)

- Caso queira exibir o número pessoal ou corporativo da sua empresa, é necessário validá-lo via **OTP** ou chamada curta.
- Esse número **não recebe chamadas inbound da IA** — ele serve apenas como identificação no Caller ID.
- **Atenção:** A verificação por OTP é exigida **somente** quando você usa um número próprio como Caller ID.

## 3. Integração SIP

- Se você utiliza um PBX ou plataforma VoIP, é possível enviar um Caller ID personalizado a partir do seu próprio sistema.
- Certifique-se de que o **trunk SIP** ou o provedor permitem a apresentação do Caller ID configurado.

## 4. Regras por País

- Alguns países não permitem o uso de números que você não possui como Caller ID.
- Sempre garanta que você tem **direito legal** de exibir o número para evitar bloqueios, marcação como spam ou problemas regulatórios.

---

**Dica:** Antes de iniciar uma campanha em grande escala, faça testes ligando para o seu próprio telefone ou para alguém de confiança para confirmar que o Caller ID está sendo exibido corretamente.
