## Como começar

<Steps>
  <Step title="Alugar um número de telefone">
    Alugue um número de telefone que será utilizado para realizar e receber chamadas
  </Step>
  <Step title="Criar um assistente">
    - Ajuste o prompt do sistema (o prompt do sistema é o conjunto de instruções que o assistente utiliza para conduzir a conversa) - Adicione ferramentas (ferramentas são ações que o assistente pode executar, por exemplo, integração com o Cal.com para agendamento de reuniões) - Defina idioma, voz e outros parâmetros
  </Step>
  <Step title="Criar uma campanha">
    Defina os parâmetros da campanha, como intervalo de horário de chamadas, dias da semana, intervalo de tentativas, entre outros
  </Step>
  <Step title="Adicionar leads">
    - Adicione leads à campanha. Leads são os números de telefone que receberão as chamadas. - É possível importar leads a partir de um arquivo CSV ou automatizá-los utilizando a plataforma de automação no-code (para receber leads de CRMs, formulários e outros). - Você pode adicionar variáveis aos leads para personalizar a conversa.
  </Step>
  <Step title="Executar a campanha">
    - Execute a campanha e aguarde os resultados. - Você pode acompanhar a campanha em tempo real. - É possível exportar os resultados para um arquivo CSV.
  </Step>
</Steps>
