# Fundamentos da Plataforma

## Tipos de Chamadas

1. Chamadas de Entrada: Chamadas recebidas pelo sistema
2. Chamadas de Saída: Chamadas iniciadas pelo sistema

## Campanhas

Campanhas são conjuntos organizados de chamadas com objetivos específicos e públicos-alvo definidos.

## Leads

Leads são potenciais clientes ou contatos que são trabalhados em uma campanha.

## Configuração do Assistente

- Engine: Modelo de IA que conduz a conversa
- Prompt: Instruções iniciais para o assistente de IA
- Mensagem Inicial: Primeira mensagem falada pelo assistente
- Idioma: Idioma utilizado na conversa
- Voz: Voz selecionada para o assistente
- Ferramentas: Funcionalidades adicionais disponíveis para o assistente
- Extrações Pós-Chamada: Dados extraídos após a chamada para análise

## Plataforma de Automação

Plataforma no-code para configurar e gerenciar chamadas automatizadas.

## Provisionamento de Números Telefônicos

Processo de aquisição e configuração de números telefônicos para uso em campanhas.

## Custos

- Custo por minuto de chamadas automatizadas: Cobrança pelo uso do serviço
- Custo do provedor de telefonia: Tarifas adicionais pelo uso dos serviços de comunicação

## Ferramentas

- Transferir Chamada: Capacidade de transferir a chamada para um agente humano
- Encerrar Chamada: Opção para finalizar a chamada
- Integração de Calendário Cal.com: Funcionalidade de agendamento integrada ao Cal.com
