---
title: "Excluir chamada"
api: "DELETE /user/calls/{call}"
description: "Exclui um registro de chamada específico"
---

Este endpoint permite excluir um registro de chamada específico que pertença ao usuário autenticado.

### Parâmetros de Caminho

<ParamField path="call" type="integer" required>
  Identificador único da chamada a ser excluída
</ParamField>

### Resposta

<ResponseField name="message" type="string">
  Mensagem de confirmação indicando que a chamada foi excluída com sucesso
</ResponseField>

### Respostas de Erro

<ResponseField name="404 Not Found">
  <Expandable title="Resposta de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro quando a chamada não é encontrada ou não pertence ao usuário autenticado
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Success Response
{
  "message": "Chamada excluída com sucesso"
}
{
  "message": "Chamada não encontrada"
}
</ResponseExample>
Observações
Apenas chamadas que pertençam a assistentes do usuário autenticado podem ser excluídas

Após a exclusão, a chamada não poderá ser recuperada

Esta ação removerá permanentemente o registro da chamada, incluindo qualquer transcrição, gravação e metadados associados

O arquivo de gravação da chamada (se existir) também será removido do armazenamento
