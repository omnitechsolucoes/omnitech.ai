---
title: "Atualizar status da campanha"
api: "POST /user/campaigns/update-status"
description: "Inicia ou interrompe uma campanha no sistema OmniTech"
---

Este endpoint permite iniciar ou interromper uma campanha no sistema OmniTech.

### Corpo da Requisição

<ParamField body="campaign_id" type="integer" required>
  ID da campanha a ser atualizada
</ParamField>

<ParamField body="action" type="string" required>
  A ação a ser executada na campanha. Deve ser "start" ou "stop"
</ParamField>

### Resposta

<ResponseField name="message" type="string" default="Campaign started successfully.">
  Mensagem da resposta
</ResponseField>

<ResponseField name="success" type="boolean" default="true">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="data" type="object">
  Dados da resposta
  
  <Expandable title="propriedades de data">
    <ResponseField name="campaign_id" type="integer">
      ID da campanha atualizada
    </ResponseField>
    
    <ResponseField name="status" type="string" default="in-progress">
      Novo status da campanha (ex.: "in-progress" para campanhas iniciadas, "stopped" para campanhas interrompidas)
    </ResponseField>
  </Expandable>
</ResponseField>
