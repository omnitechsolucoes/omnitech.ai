---
title: "Obter ferramenta de uso durante a chamada"
api: "GET /user/tools/{id}"
description: "Recupera uma ferramenta específica de uso durante a chamada pelo ID"
---

Este endpoint permite recuperar informações detalhadas sobre uma ferramenta específica de uso durante a chamada.

### Headers

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Deve ser `application/json`
</ParamField>

<ParamField header="Accept" type="string" required>
  Deve ser `application/json`
</ParamField>

### Parâmetros de Caminho

<ParamField path="id" type="integer" required>
  Identificador único da ferramenta
</ParamField>

### Campos da Resposta

<ResponseField name="id" type="integer">
  Identificador único da ferramenta
</ResponseField>

<ResponseField name="name" type="string">
  Nome da ferramenta (minúsculas com underscore)
</ResponseField>

<ResponseField name="description" type="string">
  Explicação detalhada de quando e como a IA deve utilizar esta ferramenta
</ResponseField>

<ResponseField name="endpoint" type="string">
  URL do endpoint da API que será chamado
</ResponseField>

<ResponseField name="method" type="string">
  Método HTTP (GET, POST, PUT, PATCH, DELETE)
</ResponseField>

<ResponseField name="timeout" type="integer">
  Timeout da requisição em segundos (1-30)
</ResponseField>

<ResponseField name="headers" type="array">
  Headers HTTP enviados na requisição
  <Expandable title="propriedades de headers">
    <ResponseField name="name" type="string">
      Nome do header
    </ResponseField>
    <ResponseField name="value" type="string">
      Valor do header
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="schema" type="array">
  Parâmetros que a IA irá extrair e enviar para o endpoint
  <Expandable title="propriedades de schema">
    <ResponseField name="name" type="string">
      Nome do parâmetro
    </ResponseField>
    <ResponseField name="type" type="string">
      Tipo do parâmetro (string, number, boolean)
    </ResponseField>
    <ResponseField name="description" type="string">
      Descrição para ajudar a IA a entender como extrair este parâmetro
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="created_at" type="string">
  Timestamp ISO 8601 de criação da ferramenta
</ResponseField>

<ResponseField name="updated_at" type="string">
  Timestamp ISO 8601 da última atualização da ferramenta
</ResponseField>
