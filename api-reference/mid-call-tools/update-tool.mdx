---
title: "Atualizar ferramenta de uso durante a chamada"
api: "PUT /user/tools/{id}"
description: "Atualiza uma ferramenta de uso durante a chamada existente"
---

Este endpoint permite atualizar uma ferramenta de uso durante a chamada já existente. Todos os campos são opcionais — informe apenas os campos que deseja alterar.

### Parâmetros de Caminho

<ParamField path="id" type="integer" required>
  Identificador único da ferramenta a ser atualizada
</ParamField>

### Parâmetros do Corpo

<ParamField body="name" type="string" optional>
  Nome da ferramenta — deve conter apenas letras minúsculas e underscores, e iniciar com uma letra
</ParamField>

<ParamField body="description" type="string" optional>
  Explicação detalhada de quando e como a IA deve utilizar esta ferramenta (máx. 255 caracteres)
</ParamField>

<ParamField body="endpoint" type="string" optional>
  URL válida do endpoint da API a ser chamada
</ParamField>

<ParamField body="method" type="string" optional>
  Método HTTP: `GET`, `POST`, `PUT`, `PATCH` ou `DELETE`
</ParamField>

<ParamField body="timeout" type="integer" optional>
  Tempo limite da requisição em segundos (1–30)
</ParamField>

<ParamField body="headers" type="array" optional>
  Headers HTTP enviados na requisição (substitui os headers existentes)
  <Expandable title="propriedades de headers">
    <ParamField body="name" type="string" required>
      Nome do header
    </ParamField>
    <ParamField body="value" type="string" required>
      Valor do header
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="schema" type="array" optional>
  Esquema de parâmetros (substitui o schema existente)
  <Expandable title="propriedades do schema">
    <ParamField body="name" type="string" required>
      Nome do parâmetro (2–32 caracteres)
    </ParamField>
    <ParamField body="type" type="string" required>
      Tipo do parâmetro: `string`, `number` ou `boolean`
    </ParamField>
    <ParamField body="description" type="string" required>
      Descrição do parâmetro (3–255 caracteres)
    </ParamField>
  </Expandable>
</ParamField>

### Campos da Resposta

<ResponseField name="message" type="string">
  Mensagem de sucesso
</ResponseField>

<ResponseField name="data" type="object">
  Objeto da ferramenta atualizada com todos os valores atuais
</ResponseField>
