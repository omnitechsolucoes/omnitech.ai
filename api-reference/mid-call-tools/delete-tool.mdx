---
title: "Excluir ferramenta de uso durante a chamada"
api: "DELETE /user/tools/{id}"
description: "Exclui permanentemente uma ferramenta de uso durante a chamada"
---

Este endpoint permite excluir permanentemente uma ferramenta de uso durante a chamada. Isso também removerá a ferramenta de todos os assistentes aos quais ela estiver atribuída.

### Headers

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Deve ser `application/json`
</ParamField>

<ParamField header="Accept" type="string" required>
  Deve ser `application/json`
</ParamField>

### Parâmetros de Caminho

<ParamField path="id" type="integer" required>
  Identificador único da ferramenta a ser excluída
</ParamField>

### Campos da Resposta

<ResponseField name="message" type="string">
  Mensagem de sucesso confirmando a exclusão
</ResponseField>

<ResponseExample>

```json 200 OK
{
  "message": "Ferramenta excluída com sucesso"
}
{
  "message": "Ferramenta não encontrada"
}
</ResponseExample>
Observação
Quando uma ferramenta é excluída, ela é automaticamente removida de todos os assistentes. Para gerenciar atribuições de ferramentas sem excluí-las, utilize:

Atualizar Assistente – Utilize o parâmetro tool_ids para remover ferramentas de assistentes específicos
