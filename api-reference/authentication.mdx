A API da OmniTech utiliza chaves de API para autenticar as requisições. Você precisa incluir sua chave de API no header de cada requisição feita à API.

## Obtendo uma Chave de API

Para obter uma chave de API:

1. Faça login na sua conta da OmniTech em [https://app.omnitechsolucoes.com.br](https://app.omnitechsolucoes.com.br)
2. Navegue até a página "API Keys"
3. Clique no botão "Criar nova API Key"
4. Copie sua nova chave de API e armazene-a de forma segura

<Warning>
  Mantenha sua chave de API confidencial e segura. Não a compartilhe publicamente nem a inclua
  em sistemas de controle de versão.
</Warning>

## Usando a Chave de API

Inclua sua chave de API no header `Authorization` de cada requisição:

Authorization: Bearer YOUR_API_KEY


Substitua `YOUR_API_KEY` pela sua chave de API real.

## Exemplo de Requisição

Abaixo um exemplo de como incluir a chave de API em uma requisição cURL:

```bash
curl -X GET "https://api.omnitechsolucoes.com.br/api/user/campaigns" \
     -H "Authorization: Bearer YOUR_API_KEY"
Substitua YOUR_API_KEY pela sua chave de API real ao realizar as requisições.
