---
title: "Desativar webhook do assistente"
api: "POST /user/assistants/disable-webhook"
description: "Desativa as notificações de webhook para um assistente específico"
---

Este endpoint desativa as notificações de webhook de um assistente, interrompendo todos os envios de atualizações em tempo real sobre conclusões de chamadas e dados.

### Corpo da Requisição

<ParamField body="assistant_id" type="integer" required>
  O ID do assistente para o qual o webhook será desativado
</ParamField>

### Resposta

<ResponseField name="message" type="string">
  Mensagem de sucesso confirmando que o webhook foi desativado
</ResponseField>

<ResponseField name="data" type="array">
  Array vazio (reservado para uso futuro)
</ResponseField>

### Respostas de Erro

<ResponseField name="404 Not Found">
  <Expandable title="Resposta de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro quando o assistente não é encontrado ou não pertence ao usuário autenticado
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="422 Validation Error">
  <Expandable title="Resposta de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro indicando falha de validação
    </ResponseField>
    <ResponseField name="errors" type="object">
      Erros detalhados de validação para cada campo
    </ResponseField>
  </Expandable>
</ResponseField>

Comportamento do Webhook
Quando o webhook é desativado:

O assistente deixará de enviar requisições HTTP POST para a URL do webhook

Tanto is_webhook_active será definido como false quanto webhook_url será definido como null

Quaisquer chamadas em andamento serão concluídas normalmente, porém nenhuma notificação de webhook será enviada

Os dados das chamadas continuarão disponíveis através dos endpoints de chamadas da API

Observações
O assistente deve pertencer ao usuário autenticado

Esta ação remove completamente a configuração de webhook do assistente

Você pode reativar os webhooks posteriormente utilizando o endpoint de habilitar webhook com uma nova URL

Desativar webhooks não afeta a gravação de chamadas nem a coleta de dados
