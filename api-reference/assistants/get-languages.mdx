---
title: "Obter idiomas disponíveis"
api: "GET /user/assistants/languages"
description: "Recupera todos os idiomas disponíveis para configuração de assistentes"
---

Este endpoint retorna uma lista de todos os idiomas disponíveis que podem ser utilizados ao criar ou atualizar assistentes.

### Campos da Resposta

<ResponseField name="data" type="array">
  <Expandable title="propriedades">
    <ResponseField name="id" type="integer">
      Identificador único do idioma
    </ResponseField>

    <ResponseField name="name" type="string">
      Nome de exibição do idioma
    </ResponseField>

    <ResponseField name="iso_2" type="string">
      Código do idioma em duas letras no padrão ISO 639-1
    </ResponseField>
  </Expandable>
</ResponseField>

Observações
Este endpoint não requer nenhum parâmetro

Utilize o campo name ao criar ou atualizar assistentes

O código iso_2 pode ser útil para funcionalidades específicas por idioma

Todos os idiomas disponíveis são retornados em uma única requisição
