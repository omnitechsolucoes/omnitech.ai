---
title: "Obter assistentes outbound"
api: "GET /user/assistants/outbound"
description: "Recupera todos os assistentes outbound do usuário autenticado"
---

Este endpoint retorna uma lista de todos os assistentes outbound pertencentes ao usuário autenticado. Assistentes outbound são utilizados para realizar chamadas para leads e prospects.

### Campos da Resposta

<ResponseField name="data" type="array">
  <Expandable title="propriedades">
    <ResponseField name="id" type="integer">
      Identificador único do assistente
    </ResponseField>

    <ResponseField name="user_id" type="integer">
      ID do usuário proprietário do assistente
    </ResponseField>

    <ResponseField name="name" type="string">
      Nome do assistente
    </ResponseField>

    <ResponseField name="type" type="string">
      Tipo do assistente (sempre "outbound" para este endpoint)
    </ResponseField>

    <ResponseField name="mode" type="string">
      Modo da engine (`pipeline` ou `multimodal`)
    </ResponseField>

    <ResponseField name="status" type="string">
      Status atual do assistente
    </ResponseField>

    <ResponseField name="voice_id" type="integer">
      ID da voz utilizada pelo assistente
    </ResponseField>

    <ResponseField name="language_id" type="integer">
      ID do idioma utilizado pelo assistente
    </ResponseField>

    <ResponseField name="timezone" type="string">
      Configuração de fuso horário do assistente
    </ResponseField>

    <ResponseField name="initial_message" type="string">
      Mensagem inicial que o assistente irá falar
    </ResponseField>

    <ResponseField name="system_prompt" type="string">
      Prompt de sistema que define o comportamento do assistente
    </ResponseField>

    <ResponseField name="variables" type="object">
      Variáveis personalizadas definidas para o assistente
    </ResponseField>

    <ResponseField name="webhook_url" type="string">
      URL do webhook para notificações pós-chamada
    </ResponseField>

    <ResponseField name="is_webhook_active" type="boolean">
      Indica se as notificações de webhook estão habilitadas
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Data e hora de criação do assistente
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Data e hora da última atualização do assistente
    </ResponseField>
  </Expandable>
</ResponseField>

Observações
Este endpoint retorna apenas assistentes do tipo "outbound"

Todos os assistentes pertencentes ao usuário autenticado são retornados (sem paginação)

Utilize esses assistentes em conjunto com o endpoint de realização de chamadas para iniciar chamadas outbound

Assistentes outbound podem ser utilizados para vendas, follow-ups, pesquisas e outros cenários de chamadas ativas
