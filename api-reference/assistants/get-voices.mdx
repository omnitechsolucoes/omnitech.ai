---
title: "Obter vozes disponíveis"
api: "GET /user/assistants/voices"
description: "Recupera todas as vozes disponíveis para configuração de assistentes"
---

Este endpoint retorna uma lista de todas as vozes disponíveis que podem ser utilizadas ao criar ou atualizar assistentes, com opção de filtragem por modo do assistente.

### Parâmetros de Consulta

<ParamField query="mode" type="string" optional>
  Filtra as vozes pelo modo do assistente. Opções: `pipeline`, `multimodal`
</ParamField>

### Campos da Resposta

<ResponseField name="data" type="array">
  <Expandable title="propriedades">
    <ResponseField name="id" type="integer">
      Identificador único da voz
    </ResponseField>

    <ResponseField name="name" type="string">
      Nome de exibição da voz
    </ResponseField>

    <ResponseField name="mode" type="string">
      Modo do assistente com o qual esta voz é compatível (`pipeline` ou `multimodal`)
    </ResponseField>
  </Expandable>
</ResponseField>

Observações
Utilize o campo id ao criar ou atualizar assistentes

O campo mode indica com qual engine de assistente a voz é compatível

Filtre por modo para obter apenas vozes compatíveis com o tipo de assistente desejado

Caso nenhum filtro de modo seja informado, todas as vozes disponíveis serão retornadas

Apenas vozes públicas ou pertencentes ao usuário autenticado são retornadas
