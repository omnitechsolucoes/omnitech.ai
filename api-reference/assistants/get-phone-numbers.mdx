---
title: "Obter números de telefone disponíveis"
api: "GET /user/assistants/phone-numbers"
description: "Recupera todos os números de telefone disponíveis para atribuição a assistentes"
---

Este endpoint retorna uma lista de números de telefone pertencentes ao usuário autenticado que podem ser atribuídos a assistentes, com opção de filtragem por tipo de assistente.

### Parâmetros de Consulta

<ParamField query="type" type="string" optional>
  Filtra os números de telefone pelo tipo de assistente. Opções: `inbound`, `outbound`
</ParamField>

### Campos da Resposta

<ResponseField name="data" type="array">
  <Expandable title="propriedades">
    <ResponseField name="id" type="integer">
      Identificador único do número de telefone
    </ResponseField>

    <ResponseField name="phone_number" type="string">
      Número de telefone no formato E.164
    </ResponseField>

    <ResponseField name="country_code" type="string">
      Código do país do número de telefone
    </ResponseField>

    <ResponseField name="type_label" type="string">
      Rótulo do tipo do número de telefone em formato legível
    </ResponseField>

    <ResponseField name="is_available" type="boolean">
      Indica se o número de telefone está disponível para atribuição (não está atualmente atribuído a um assistente inbound)
    </ResponseField>
  </Expandable>
</ResponseField>

Observações
Apenas números de telefone pertencentes ao usuário autenticado são retornados

Quando type=inbound é especificado, números do tipo caller ID e números já atribuídos são filtrados

O campo is_available indica se o número pode ser atribuído a um novo assistente inbound

Os números de telefone são retornados no formato de coleção de valores (array indexado)

Utilize o campo id ao atribuir números de telefone a assistentes
