---
title: "Obter modelos LLM disponíveis"
api: "GET /user/assistants/models"
description: "Recupera todos os modelos LLM disponíveis para configuração de assistentes"
---

Este endpoint retorna uma lista de todos os Modelos de Linguagem de Grande Escala (LLMs) disponíveis que podem ser utilizados ao criar ou atualizar assistentes.

### Campos da Resposta

<ResponseField name="data" type="array">
  <Expandable title="propriedades">
    <ResponseField name="id" type="integer">
      Identificador único do modelo LLM
    </ResponseField>

    <ResponseField name="name" type="string">
      Nome de exibição do modelo LLM
    </ResponseField>
  </Expandable>
</ResponseField>

Observações
Este endpoint não requer nenhum parâmetro

Utilize o campo name ao criar ou atualizar assistentes

Todos os modelos LLM disponíveis são retornados em uma única requisição

Diferentes modelos podem apresentar capacidades, desempenho e custos distintos
