---
title: "Excluir assistente"
api: "DELETE /user/assistant/{id}"
description: "Exclui um assistente de IA específico"
---

Este endpoint permite excluir permanentemente um assistente de IA que pertença ao usuário autenticado.

### Parâmetros de Caminho

<ParamField path="id" type="integer" required>
  Identificador único do assistente a ser excluído
</ParamField>

### Resposta

<ResponseField name="message" type="string">
  Mensagem de confirmação indicando que o assistente foi excluído com sucesso
</ResponseField>

### Respostas de Erro

<ResponseField name="404 Not Found">
  <Expandable title="Resposta de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro quando o assistente não é encontrado ou não pertence ao usuário autenticado
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="500 Internal Server Error">
  <Expandable title="Resposta de Erro">
    <ResponseField name="message" type="string">
      Mensagem de erro indicando falha na exclusão do assistente
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Success Response
{
  "message": "Assistente excluído com sucesso"
}
{
  "message": "Assistente não encontrado"
}
{
  "message": "Falha na exclusão do assistente"
}
</ResponseExample>
Observações
Apenas assistentes que pertençam ao usuário autenticado podem ser excluídos

Após a exclusão, o assistente não poderá ser recuperado

Esta ação removerá permanentemente o assistente e todas as suas configurações

Se o assistente estiver lidando com chamadas ativas no momento da exclusão, essas chamadas poderão ser afetadas
