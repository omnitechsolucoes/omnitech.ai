## Criando um novo app

Você precisa criar um aplicativo no LinkedIn para permitir que usuários façam login no seu site usando suas contas do LinkedIn. Para criar um novo app, siga os passos abaixo:

- Acesse o site de desenvolvedores do LinkedIn: https://developer.linkedin.com/  
- Clique no botão **“Create App”** no banner.
- Para criar corretamente o aplicativo, preencha as seções solicitadas, conforme explicado abaixo.

## Criar um app

- **Application Name** – Informe o nome do seu aplicativo.  
- **Company name** – Selecione uma empresa existente ou escolha **New Company**. Caso escolha uma nova empresa, será solicitado o nome dela.  
- **Description** – Informe uma descrição para o aplicativo.  
- **Business Email** – Informe seu endereço de e-mail.  
- **Legal terms** – Leia os termos de uso e marque a caixa confirmando que leu e concorda com os termos.

Clique no botão de projeto no menu superior e você verá o pop-up **Select a project**. Clique em **NEW PROJECT** para criar um projeto.

## Produtos

Após clicar em **Create app**, você será redirecionado para a aba **Products**.

   <Frame>
     <img
       src="/resources/linkedin-products-tab.png"
       alt="LinkedIn Products Tab"
     />
   </Frame>

Role a lista de produtos até encontrar **Sign In with LinkedIn using OpenID Connect** e clique no botão **Request Access**.

   <Frame>
     <img
       src="/resources/linkedin-openid-connect.png"
       alt="LinkedIn OpenID Connect"
     />
   </Frame>

Leia os termos; se concordar, marque a caixa de confirmação e clique em **Request Access**.

   <Frame>
     <img
       src="/resources/linkedin-request-acces.png"
       alt="LinkedIn Request Access"
     />

Acesse a aba **Auth** e adicione as URLs de redirecionamento.

   <Frame>

## Configurações OAuth 2.0

Nas URLs de redirecionamento, basta informar o endereço do seu site, por exemplo:  
`https://seusite.com/login/linkedin-openid/callback`
